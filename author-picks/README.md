# Author Picks

This directory contains curated selections of the best and most useful prompts, hand-picked by the repository maintainers.

## Purpose

- Highlight quality content for users who don't want to browse everything
- Provide "best of" lists for common use cases
- Enable maintainers to recommend specific prompts

## Structure

```
author-picks/
├── README.md            # This file
├── best-for-beginners.md    # Recommended starting points
├── most-versatile.md         # Highly adaptable prompts
├── newest-additions.md       # Recent quality additions
└── curated/
    └── [additional curated lists]
```

## Picking Criteria

A prompt qualifies for author picks if it:

1. Has complete frontmatter and content
2. Solves a common, real problem
3. Has clear, actionable instructions
4. Includes good examples
5. Has been tested in real scenarios

## Categories

| List | Description |
|------|-------------|
| best-for-beginners | Easy to understand, high value prompts for newcomers |
| most-versatile | Can be applied to many different situations |
| newest-additions | Recently added prompts worth attention |

---

**Note**: This directory may be empty initially. As prompts are tested and evaluated, curated lists will be populated.
