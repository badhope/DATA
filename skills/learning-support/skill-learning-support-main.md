# Learning Support Skill

个性化学习支持与知识巩固的完整指南。

## 学习水平评估

### 水平定义

| 级别 | 名称 | 描述 | 能力特征 |
|------|------|------|----------|
| L1 | 零基础 | 完全不了解 | 了解基本概念术语 |
| L2 | 入门 | 有初步了解 | 能理解简单应用 |
| L3 | 初级 | 能独立使用 | 能完成基础任务 |
| L4 | 中级 | 有实践经验 | 能独立解决问题 |
| L5 | 高级 | 理解原理 | 能优化和创新 |
| L6 | 专家 | 精通全面 | 能传授和指导 |

### 自我评估问卷

```markdown
## 机器学习水平自评

### 基础概念
1. 你能解释什么是监督学习和无监督学习吗？
   - ❌ 完全不了解 → L1
   - ✅ 知道但无法举例 → L2
   - ✅ 能举例说明 → L3

2. 你能说出至少3种常见的机器学习算法吗？
   - ❌ 不能 → L1-L2
   - ✅ 能 → L3+

### 实践能力
3. 你是否独立完成过机器学习项目？
   - ❌ 没有 → L1-L3
   - ✅ 完成过1-2个 → L4
   - ✅ 完成过多个复杂项目 → L5+

### 理论深度
4. 你能解释过拟合和欠拟合的成因吗？
   - ❌ 不能 → L1-L2
   - ✅ 知道但不深入 → L3
   - ✅ 能解释并提出解决方案 → L4+

---

## 评估结果
综合水平: L3 (初级)
建议起点: 从"实战基础"模块开始
```

## 学习方法论

### 费曼学习法

```markdown
## 费曼学习法四步

### Step 1: 选择一个概念
选择一个你想学习的概念/主题

### Step 2: 把它教给小白
用最简单的话解释给"8岁小孩"听

### Step 3: 发现卡点
回到原始资料，弄清楚卡点在哪里

### Step 4: 简化类比
用生活化的类比来解释复杂概念

---

## 示例: 解释"递归"

### ❌ 糟糕的解释
"递归就是函数调用自身的过程"

### ✅ 费曼式的解释
想象俄罗斯套娃，每个娃娃里面都有一个更小的娃娃，直到最小的那个。

或者想象照镜子，镜子里的人在照镜子...

### 代码示例
```python
def countdown(n):
    print(n)
    if n > 0:
        countdown(n-1)  # 调用自己
```
```

### 闪卡模板
```
正面: 什么是闭包？

背面: 闭包是指一个函数能够访问其词法作用域外部的变量。
      即使外部函数已经返回，内部函数仍然可以访问这些变量。

示例:
def outer():
    x = 10
    def inner():
        print(x)  # 访问了外部变量
    return inner
```
```

## 学习资源推荐

### 资源类型

| 类型 | 特点 | 适用场景 |
|------|------|----------|
| 官方文档 | 权威、准确 | 深入学习、查阅 |
| 教程/课程 | 系统、循序渐进 | 入门学习 |
| 视频 | 直观、易理解 | 概念理解 |
| 书籍 | 全面、深入 | 系统学习 |
| 实践项目 | 动手、巩固 | 技能提升 |
| 社区/论坛 | 交流、答疑 | 问题解决 |
