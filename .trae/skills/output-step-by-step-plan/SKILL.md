# Output as Step-by-Step Plan

将任务输出为分步骤的计划格式。

## When to Use

- 任务分解和规划
- 执行计划制定
- 操作步骤整理
- 项目阶段划分

## Plan Structure

1. 步骤 1：准备阶段
2. 步骤 2：实施阶段
3. 步骤 3：验证阶段

## Related Prompts

- `output-as-step-by-step-plan` - 步骤计划
- `output-as-checklist` - 检查清单
- `output-as-markdown-report` - 报告格式
