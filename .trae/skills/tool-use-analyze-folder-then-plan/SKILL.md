# Analyze Folder Then Plan

先分析文件夹结构再制定执行计划。

## When to Use

- 新项目需要了解结构
- 制定修改计划前
- 定位问题文件前
- 制定代码生成计划前

## Process

1. 分析文件夹结构
2. 识别关键文件和目录
3. 理解模块划分
4. 制定行动计划

## Related Prompts

- `tool-use-read-files-first` - 先读取文件
- `tool-use-search-before-concluding` - 下结论前搜索
