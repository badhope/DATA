# Combine Multiple Tool Results

整合来自不同工具的输出形成完整答案。

## When to Use

- 需要整合多个文件的信息
- 命令输出需要关联分析
- 多源信息综合判断
- 形成完整分析报告

## Process

1. 收集各工具输出
2. 识别信息关联
3. 整合形成完整答案
4. 验证信息一致性

## Related Prompts

- `tool-use-read-files-first` - 先读取文件
- `tool-use-search-before-concluding` - 下结论前搜索
