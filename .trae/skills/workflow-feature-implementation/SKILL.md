# Feature Implementation Workflow

多步骤功能开发的完整工作流。

## When to Use

- 实现新功能
- 从需求出发生成代码
- 需要遵循项目规范
- 功能开发与验证

## Workflow Steps

1. **理解需求** - 分析功能需求和验收标准
2. **创建执行计划** - 分解任务为可执行步骤
3. **分解子任务** - 划分模块和依赖
4. **实现功能** - 按计划编码实现
5. **代码审查** - 检查质量和规范
6. **测试验证** - 确认功能正确性
