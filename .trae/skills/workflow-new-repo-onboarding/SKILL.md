# New Repo Onboarding Workflow

新仓库快速上手的完整工作流。

## When to Use

- 首次接触陌生代码库
- 需要快速理解项目结构
- 加入新项目团队
- 接手遗留代码

## Workflow Steps

1. **分析仓库结构** - 了解目录组织和模块划分
2. **总结架构** - 理解核心组件和依赖关系
3. **定位关键文件** - 找到入口点和配置文件
4. **理解依赖** - 分析外部依赖和构建方式

## Related Prompts

- `prompt-task-repo-analysis-analyze-repository-structure` - 仓库结构分析
- `prompt-task-repo-analysis-summarize-project-architecture` - 架构总结
- `prompt-task-repo-analysis-locate-bug-related-files` - 定位相关文件
