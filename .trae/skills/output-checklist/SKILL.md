# Output as Checklist

将任务或检查项输出为清单格式。

## When to Use

- 任务检查清单
- 测试用例列表
- 审核项目列表
- 待办事项整理

## Checklist Format

- [ ] 项目 1
- [ ] 项目 2
- [ ] 项目 3

## Related Prompts

- `output-as-checklist` - 检查清单
- `output-as-step-by-step-plan` - 步骤计划
- `output-as-comparison-table` - 对比表格
