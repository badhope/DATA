# Search Before Concluding

在下结论前充分搜索和验证信息。

## When to Use

- 得出技术结论前
- 提供建议前
- 解决问题前
- 评估方案前

## Process

1. 搜索相关信息
2. 验证多个信息源
3. 分析搜索结果
4. 得出结论
