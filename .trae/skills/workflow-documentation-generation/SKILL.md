# Documentation Generation Workflow

自动生成技术文档的工作流。

## When to Use

- 生成项目文档
- API 文档编写
- README 更新
- 注释文档化

## Workflow Steps

1. **分析需求** - 确定文档类型和范围
2. **收集信息** - 从代码和配置中提取
3. **生成内容** - 按模板生成文档
4. **审核优化** - 检查准确性和完整性

## Related Prompts

- `output-as-markdown-report` - Markdown 报告格式
- `prompt-task-repo-analysis-summarize-project-architecture` - 架构文档
