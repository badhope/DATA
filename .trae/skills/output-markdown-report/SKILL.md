# Output as Markdown Report

将数据输出为格式化的 Markdown 报告。

## When to Use

- 分析报告生成
- 项目总结文档
- 技术说明文档
- 状态报告输出

## Report Structure

- 标题和概述
- 主要内容章节
- 数据表格和列表
- 结论和建议
