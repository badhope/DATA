# Examples

This directory contains real-world examples of how to use the prompts in this repository.

## Purpose

- Demonstrate prompt usage in realistic scenarios
- Show expected inputs and outputs
- Help users understand when and how to apply prompts

## Structure

```
examples/
├── coding/              # Code generation examples
├── debugging/           # Debugging workflow examples
├── repo-analysis/       # Repository analysis examples
├── planning/            # Planning examples
├── general/             # General capability examples
├── creative-special/    # Creative writing examples
├── workflow/            # Multi-step workflow examples
└── README.md            # This file
```

## Adding Examples

1. Create a file in the appropriate category directory
2. Follow the example template in EXTENSION-GUIDE.md
3. Include realistic input/output
4. Add frontmatter with metadata

## Categories

| Category | Description |
|----------|-------------|
| coding | Code generation, implementation examples |
| debugging | Bug investigation and fix examples |
| repo-analysis | Project understanding examples |
| planning | Task breakdown examples |
| general | Clarification, reasoning examples |
| creative-special | Creative writing examples |
| workflow | Multi-step process examples |
