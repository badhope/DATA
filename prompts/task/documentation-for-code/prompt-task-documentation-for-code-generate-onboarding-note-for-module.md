# Context

新开发者入门指南帮助新成员快速理解代码模块，降低学习曲线，提高团队协作效率。一份好的入门指南应该循序渐进，从基础概念到实际应用，让新开发者能快速上手工作。本 prompt 的核心目标是：**指导 AI 为代码模块生成结构清晰、内容完整的入门指南**。

# Usage Notes

1. **循序渐进**：从基础到复杂
2. **实践导向**：多提供练习任务
3. **及时反馈**：提供获取帮助的渠道
4. **团队融入**：帮助新开发者融入团队
5. **持续更新**：随着模块演进更新指南
