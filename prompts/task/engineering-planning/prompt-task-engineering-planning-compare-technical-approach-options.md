# Context

技术方案比较是项目规划中的重要决策环节。在多个可行方案中选择最优方案需要基于全面的评估维度和客观的分析。本 prompt 的核心目标是：**指导 AI 系统化地比较多技术方案的优劣，给出明确的决策建议**。

# Variables

| 变量 | 说明 | 示例 |
|------|------|------|
| `problem_statement` | 问题陈述 | `"需要选择消息队列方案"` |
| `options` | 技术方案 | `[{name: "Kafka", pros: [...], cons: [...]}]` |
| `criteria` | 评估维度 | `["性能", "可靠性", "成本"]` |

# Usage Notes

1. **评估维度全面**：覆盖技术、商业、团队等多个维度
2. **客观评分**：基于事实和数据，避免主观偏好
3. **权重合理**：根据项目特点调整权重
4. **成本考虑 TCO**：考虑全生命周期成本
5. **明确建议**：给出清晰的推荐方案和理由

# Example Input

```yaml
problem_statement: "需要选择实时数据处理方案"
options:

    description: "轻量级流处理"
```
