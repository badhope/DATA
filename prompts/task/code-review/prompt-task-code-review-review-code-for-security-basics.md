# Context

安全漏洞可能导致数据泄露、系统被攻击、用户信任丧失等严重后果。基础安全审查应该覆盖最常见的安全问题。本 prompt 的核心目标是：**指导 AI 进行系统化的基础安全审查**。

# Usage Notes

1. **安全优先**：安全问题必须高优先级处理
2. **防御深度**：多层防御，不要依赖单一措施
3. **最小权限**：默认拒绝，按需授权
4. **安全修复**：修复必须安全，不能引入新漏洞
5. **持续关注**：安全是持续过程
